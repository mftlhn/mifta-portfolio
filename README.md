# 💼 Personal Portfolio

Welcome to my personal portfolio repository.

This project is a modern portfolio website built to showcase my professional experience, technical skills, and software development projects. It also includes a simple CMS that allows me to manage portfolio content without modifying the source code.

---

## 👨‍💻 About Me

I'm a Fullstack Web Developer who enjoys building modern, scalable, and maintainable web applications. My primary expertise includes backend development with Laravel and frontend development using React and Next.js.

This portfolio serves as a central place to showcase my work, experiences, and technologies that I use in my daily development.

---

## ✨ Features

* Responsive Portfolio Website
* Portfolio CMS
* Dynamic Project Management
* Dynamic Work Experience Management
* Dynamic Skills Management
* Cloudinary Image Support
* Contact Form
* Authentication for CMS
* Supabase Integration
* Modern UI built with ShadCN UI

---

## 📸 Portfolio CMS

The built-in CMS allows me to manage portfolio content easily, including:

* Personal Information
* Skills
* Projects
* Project Images
* Work Experiences

All portfolio data is stored in Supabase, while project images are hosted on Cloudinary.
